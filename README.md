# 🎮 Retro Games - Locadora de Jogos

> **Projeto desenvolvido para a disciplina de Web Development.**

## 📖 Sobre o Projeto

A **Retro Games** é uma locadora de jogos inspirada no visual das locadoras dos anos 80 e 90, trazendo uma experiência nostálgica para os amantes de videogames clássicos.

O site foi desenvolvido com foco em apresentar um catálogo de jogos, destacar promoções e proporcionar uma navegação simples, intuitiva e com identidade visual retrô.

---

## ✨ Funcionalidades

* 🕹️ Página inicial temática.
* 🎮 Catálogo de jogos.
* 🔍 Pesquisa de títulos.
* ⭐ Destaques e lançamentos.
* 📱 Layout responsivo.
* 🎨 Interface inspirada nas locadoras clássicas.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## 🚀 Como Executar

1. Clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` em seu navegador.

Ou utilize uma extensão como **Live Server** para uma melhor experiência durante o desenvolvimento.

---

## 🎨 Tema

O projeto utiliza elementos visuais inspirados nas décadas de 1980 e 1990:

* Cores neon
* Estilo synthwave
* Fontes retrô
* Efeitos inspirados em arcades
* Atmosfera de videogames clássicos

---

## 📌 Objetivo

Este projeto foi desenvolvido para fins acadêmicos, com o objetivo de aplicar conceitos de desenvolvimento web, organização de páginas, estilização e interatividade utilizando tecnologias front-end.
